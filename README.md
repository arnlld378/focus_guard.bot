# focus_guard.bot
# 🛡️ Focus Guard Bot

**Focus Guard** — это интеллектуальный Telegram-бот на базе ИИ, который помогает фильтровать флуд в группах и пересылает только по-настоящему важные сообщения в личку администратору.

##  Основные функции
* **ИИ-фильтрация:** Использует нейросеть (GPT-4o) для анализа контекста сообщений.
* **Мгновенные уведомления:** Если сообщение признано важным, бот пересылает его вам со всеми деталями (автор, группа, текст).
* **Белый список:** Игнорирует сообщения от доверенных пользователей.
* **Работа 24/7:** Настроен для работы на облачном хостинге Render.

## 🛠️ Технологии
* **Язык:** [Python 3.10+](https://www.python.org/)
* **Библиотека:** [Aiogram 3.x](https://docs.aiogram.dev/) — для работы с Telegram API.
* **Нейросеть:** [G4F](https://github.com/xtekky/gpt4free) — бесплатный доступ к GPT-4o.
* **Деплой:** [Render.com](https://render.com/)
